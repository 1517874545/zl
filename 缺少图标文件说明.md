# 缺少图标文件说明

## 问题

小程序中缺少以下图标文件，导致控制台报错：

1. `miniprogram/assets/feature-weather.png` - 天气功能图标
2. `miniprogram/assets/feature-pomodoro.png` - 番茄钟功能图标

## 临时解决方案

我已经在代码中临时使用了现有图标：
- 天气图标：临时使用 `feature-schedule.png`（课程表图标）
- 番茄钟图标：临时使用 `feature-check.png`（打卡图标）

这样可以避免报错，但图标显示不正确。

## 永久解决方案

### 方案1：添加图标文件（推荐）

1. **准备图标文件**
   - 天气图标：建议使用云朵、太阳等天气相关图标
   - 番茄钟图标：建议使用时钟、番茄等图标
   - 尺寸建议：64x64 或 128x128 像素
   - 格式：PNG，支持透明背景

2. **放置文件**
   - 将 `feature-weather.png` 放到 `miniprogram/assets/` 目录
   - 将 `feature-pomodoro.png` 放到 `miniprogram/assets/` 目录

3. **恢复代码**
   - 在 `miniprogram/app.ts` 中恢复正确的图标路径：
     ```typescript
     iconPath: '../../assets/feature-weather.png',
     iconPath: '../../assets/feature-pomodoro.png',
     ```

### 方案2：使用在线图标

如果暂时没有图标文件，可以使用在线图标服务或图标字体。

### 方案3：使用文字图标

可以修改代码，使用文字或emoji代替图标：
- 天气：☀️ 或 🌤️
- 番茄钟：⏰ 或 🍅
